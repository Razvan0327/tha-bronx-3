# Tha Bronx 3 - Bypassin Premium

A feature-rich Lua script for Tha Bronx 3, compatible with Luarmor key system distribution.

## Features

### Main Tab
- **Player Options**: Select Player, Spectate, Bring Player, Teleport To Player, Kill/Down Player actions
- **Movement**: No Clip, Speed (adjustable), Fly (adjustable speed), Jump Power (adjustable)
- **Toggle Interfaces**: Enable/disable in-game UI elements (ThaShop, Phone, etc.)

### Money Tab
- **Farming**: Auto Farm Construction, Bank, House, Studio, Dumpsters
- **Vulnerability Section**: Generate Max Illegal Money (Manual & Auto)
- **Bank Actions**: Deposit, Withdraw, Drop with custom amounts; Auto Deposit/Withdraw/Drop
- **Duping Section**: Duplicate Current Item

### Miscellaneous Tab
- **Local Player Mods**: Infinite Stamina, Instant Respawn, Infinite Sleep/Hunger, Instant Interact, Auto Pickup Cash, Disable Blood Effects, Unlock Locked Cars, No Rent Pay, No Fall Damage, Respawn Where You Died
- **Purchase Items**: Buy items from the in-game shop
- **Teleport To Location**: Quick teleport to map locations (Basketball Court, Gun Store, Bank, etc.)
- **Outfits**: Apply preset outfits (Amiri, Nike Tech, Bape Set)

### Combat Tab
- **Silent Aim**: Toggle, keybind, visible check, aimlock type (Mouse/Camera/Closest), target parts, max distance, smoothness
- **Aimlock**: Same settings as Silent Aim with independent configuration
- **Field Of View**: FOV circle with adjustable radius and sides
- **Snapline**: Visual line to target with adjustable thickness

### Visuals Tab
- **ESP**: Enable ESP, Corner Frame ESP, Box ESP
- **Healthbar ESP**: Health Bar, Health Text, Lerp Health Color, Gradient Health
- **Cham ESP**: Chams, Thermal Effect, Visible Check, Fill/Outline Transparency
- **ESP Settings**: Distance display, Max Distance filter

### Settings Tab
- **Config Management**: Save/Load configurations, auto-load on start
- **Close Bind**: Customizable keybind to toggle UI

## Usage

1. Add this script to Luarmor with your key system
2. Execute the script in your Roblox executor
3. The GUI will appear with all tabs accessible from the sidebar
4. Configure settings and save your config in the Settings tab

## File Structure

```
src/
  main.lua    -- Main script with full GUI and feature logic
```
